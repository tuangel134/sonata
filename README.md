# Sonata

Reproductor de musica para Android: local (offline) + streaming online, con ecualizador, letras sincronizadas, Android Auto y auto-actualizacion.

## Descargar

Descarga el APK mas reciente desde la pestana **Releases** e instalalo. La primera vez, Android pedira permitir "Instalar apps desconocidas".

A partir de ahi, Sonata se actualiza sola: comprueba las releases de este repositorio al abrirse y ofrece instalar la nueva version.

## Funciones
- Biblioteca local: albumes, artistas, carpetas, listas.
- Streaming y descargas (YouTube Music via NewPipeExtractor).
- Ecualizador por bandas, bass boost, virtualizador, crossfade.
- Letras .lrc locales, embebidas y online (LRCLIB).
- Reproductor a pantalla completa con gestos y animaciones.
- Android Auto, control por voz e intents/deep links.
